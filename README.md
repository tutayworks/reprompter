## 🔧 Troubleshooting

If you run into problems, try these tips:

- **App won’t start:** Make sure your computer meets the system requirements. Restart your computer and try again.
- **Download issues:** Check your internet connection and try downloading again. Use a different browser if the download fails.
- **App crashes or freezes:** Restart the app. Close other programs that may be using a lot of memory.
- **Prompt scores very low:** Try giving more detailed answers during the interview. Avoid vague or short responses.
- **Permission errors:** On Windows or macOS, you may need to allow the app in security settings.

If problems continue, you can open an issue on the GitHub page under the “Issues” tab.
